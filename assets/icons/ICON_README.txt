# アイコン画像について

このフォルダには以下のサイズのアイコン画像を配置してください。

必要なファイル:
- icon16.png (16x16 ピクセル)
- icon48.png (48x48 ピクセル)
- icon128.png (128x128 ピクセル)

## アイコン作成のヒント

1. オンラインツールを使用:
   - Canva (https://www.canva.com/)
   - Favicon Generator (https://favicon.io/)
   - GIMP / Photoshop

2. デザイン案:
   - 弾幕（テキスト）のイメージ
   - ゲームコントローラー
   - YouTubeロゴ + シューティングゲーム要素

3. カラースキーム:
   - メインカラー: #667eea (紫)
   - アクセント: #ff6b6b (赤)
   - 背景: 透過または白

## 簡易的な代替方法

画像がない場合、Chromeは自動的にデフォルトアイコンを使用します。
拡張機能の動作には影響しません。
